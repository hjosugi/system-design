# Rate Limiter Lab

A runnable implementation drill for token bucket and sliding-window rate limiters.

## Run Simulation

```bash
python3 implementations/rate-limiter-lab/rate_limiter.py
```

## Run Tests

```bash
python3 implementations/rate-limiter-lab/test_rate_limiter.py
```

## Exercise

1. Add per-user limits.
2. Add a Redis-like storage interface.
3. Explain which limiter you would choose for login attempts, public APIs, and write-heavy ingestion.
