# Memorization

Last verified: 2026-06-20

This directory is for high-signal notes that should be easy to review before an interview.

## Memorize These Categories

- capacity estimation formulas
- latency and throughput vocabulary
- cache patterns and invalidation tradeoffs
- database replication and partitioning
- consistency models
- queue semantics
- idempotency and retry behavior
- rate limiter algorithms
- object storage concepts
- feed fanout strategies
- search/autocomplete indexing concepts
- observability signals
- failure-mode vocabulary

## Format

Each note should be short:

- concept
- when to use
- when not to use
- numbers or formulas
- failure mode
- interview phrase
- implementation drill
